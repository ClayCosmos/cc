# 红人协作平台竞品分析

> 分析日期：2026-04-09
> 更新日期：2026-04-09（修正：面向海外市场，非中国市场）

## 目标市场

**面向海外市场**。TikTok / Instagram / Facebook / YouTube 生态下的全球红人、达人、主播与机构、商家之间的协作。

## 核心发现

整个市场的最大矛盾：红人发现/分析工具 20+ 家，但"找到之后怎么协作"几乎没人做好。

现有工具要么太重（企业级 SaaS 年费 $25K+），要么太轻（Email + Google Sheets 拼凑）。

## 海外协作现状

```
建联 (TikTok/IG/FB DM)
  → 沟通 (Email + Lark/Slack)
    → 品管理 (Excel/Google Sheets)
      → 排期 (Google Calendar / 口头)
        → 交付 (Email 来回)
```

全是松散工具的拼凑，没有一个"家"。Email-based 工作流效率极低——来回邮件确认一个排期可能要 3 天。

## 海外市场 vs 中国市场的关键差异

| 维度 | 中国市场 | 海外市场 |
|------|---------|---------|
| 沟通中心 | 微信（超级 App，极难撼动） | Email + Lark/Slack（松散，无忠诚度） |
| 用户迁移成本 | 极高 | 低——本来就在多个工具间跳 |
| 达人接受新工具 | 不愿装 App | 习惯用 Web 工具，一个链接就行 |
| 平台生态锁定 | 巨量星图/蒲公英锁生态 | 无主导平台，跨 TikTok/IG/YouTube 很常见 |
| 多语言需求 | 基本单语言 | 刚需——跨境场景涉及多语种沟通 |
| WhatsApp | N/A | 部分使用但功能极少，无结构化能力 |

## 一、海外红人营销平台（重发现、轻协作）

### CreatorIQ
- **核心功能**：红人发现、关系管理、Campaign 管理、效果分析、内容审批流程
- **目标用户**：大型品牌和代理商（企业级）
- **协作方式**：内置 Campaign workflow，品牌可在平台上发 brief、审批内容、追踪交付
- **AI 能力**：AI 驱动的红人匹配和推荐、假粉检测、内容表现预测
- **定价**：企业定制报价，年费 $25K-$100K+
- **关键弱点**：价格极高；偏重发现和分析，协作层不够轻量；学习曲线陡峭

### Grin
- **核心功能**：红人关系管理、产品寄送跟踪、内容管理、支付、电商整合（Shopify 深度集成）
- **目标用户**：DTC 电商品牌
- **协作方式**：邮件整合（类似 CRM 邮件追踪）、产品 seeding 自动化、内容审批流
- **AI 能力**：有限，主要是数据分析和红人推荐
- **定价**：定制报价，通常 $1K-$3K/月起
- **关键弱点**：以邮件为核心的沟通方式比较传统；不支持即时通讯；国际化弱

### Aspire (原 AspireIQ)
- **核心功能**：红人发现、Campaign 管理、内容库、UGC 管理、联盟营销整合
- **目标用户**：中大型品牌
- **协作方式**：平台内消息、内容审批、任务看板式管理
- **AI 能力**：AI 推荐匹配、内容表现分析
- **定价**：$2K/月起
- **关键弱点**：功能全但不深；协作体验偏传统表单式

### Upfluence
- **核心功能**：红人搜索（3M+ 数据库）、Campaign 管理、联盟营销、电商整合
- **目标用户**：中型品牌和代理商
- **协作方式**：批量邮件外联、内容审批、支付管理
- **AI 能力**：AI 红人匹配、ChatGPT 整合辅助邮件撰写
- **定价**：$478/月起（Growth 计划）
- **关键弱点**：UI 偏复杂；实时协作能力弱；更像外联工具而非协作工具

### Traackr
- **核心功能**：红人发现与评估、品牌安全审查、投资回报分析、竞品监控
- **目标用户**：大型企业品牌（L'Oreal、Samsung 等）
- **协作方式**：内部团队协作为主，Campaign brief 分发
- **AI 能力**：AI 品牌适配度评分、内容分析
- **定价**：企业级，$30K+/年
- **关键弱点**：极度企业化；不适合轻量协作场景；更像分析工具

### impact.com / Creator
- **核心功能**：合作伙伴管理平台（联盟 + 红人），合同管理、支付、归因追踪
- **目标用户**：中大型品牌、注重 ROI 追踪的企业
- **协作方式**：合同驱动的工作流，自动化支付和佣金
- **AI 能力**：有限，主要在归因和欺诈检测
- **定价**：按合作伙伴数量计费
- **关键弱点**：偏联盟营销逻辑；对内容创作协作支持弱

## 二、红人 CRM / 数据工具

### Modash
- **核心功能**：红人搜索（200M+ 档案）、受众分析、监控列表、Campaign 追踪
- **目标用户**：中小品牌和代理商
- **协作方式**：红人列表管理 + 基础追踪，缺少深度协作功能
- **AI 能力**：数据驱动推荐
- **定价**：$199/月起
- **关键弱点**：本质是数据工具，不是协作工具；没有内置沟通功能

### Klear (Meltwater 旗下)
- **核心功能**：红人识别、受众分析、Campaign 管理、效果衡量
- **AI 能力**：AI 评分和推荐
- **定价**：定制报价
- **关键弱点**：被 Meltwater 收购后偏企业化；独立协作能力有限

### HypeAuditor
- **核心功能**：假粉检测（核心优势）、红人分析、市场洞察
- **AI 能力**：AI 假粉检测是其核心技术亮点
- **定价**：$299/月起
- **关键弱点**：分析强，协作弱；不解决日常沟通问题

### Julius AI / Influencer Hero
- **核心功能**：AI 驱动的红人搜索和分析
- **AI 能力**：自然语言搜索红人
- **定价**：$199/月起
- **关键弱点**：发现层 AI 强，但协作几乎为零

## 四、海外实际协作工具链

| 工具 | 角色 | 痛点 |
|------|------|------|
| Email (Gmail/Outlook) | 主要沟通渠道 | 来回慢、无结构化、难追踪状态、淹没在收件箱 |
| Lark/Slack | 部分机构内部协作 | 达人不在同一个 workspace，跨组织协作弱 |
| WhatsApp | 部分东南亚/拉美达人沟通 | 功能极少，无文件管理、无任务追踪、无结构化 |
| Google Sheets | 品/排期管理 | 手动维护，无自动化，多人编辑混乱 |
| Google Calendar | 排期 | 不关联具体合作内容，纯时间工具 |
| Notion/Airtable | 少数机构内部用 | 达人不会主动用，单向管理而非双向协作 |

## 五、市场空白

没有任何工具同时满足：

1. **邀请制** — 服务已有关系，非撮合
2. **轻量** — 达人点链接即用，零安装
3. **AI 原生** — AI 参与内容生产、沟通翻译、排期管理，不只是数据分析
4. **跨平台** — 不绑定 TikTok/IG/YouTube 某个生态
5. **双向** — 不只是品牌管理达人，而是双方协作
6. **多语言** — 跨境场景下的多语种实时沟通

## 七、最直接竞品：Grin

Grin 是唯一认真做"关系管理"而非"红人发现"的海外工具，是最接近的竞品。

| 维度 | Grin | 我们的方向 |
|------|------|---------|
| AI 内容能力 | 无 | 核心功能（brief 生成、内容脚本、生图） |
| 多语言 | 不支持 | AI 原生多语言沟通 |
| 定价 | $1-3K/月 | 更轻量、更低门槛 |
| 达人端体验 | 邮件为主，被动 | Web 端主动协作 |
| 跨境场景 | 弱 | 核心场景 |
| 入口 | 需要品牌端操作 | 一封邮件邀请，点链接即用 |

## 八、AI 原生工作空间：Moxt.ai

> 分析日期：2026-04-09

### 产品概览

Moxt 自称 **"AI-Native Workspace"**，从底层为 AI Agent 设计的工作空间，人类在其中协作。2026 年 3 月刚公开发布，处于极早期。

### 核心功能

- **momo 个人 AI 助手**：每人一个私人 AI，具备持久记忆，越用越懂用户风格和上下文
- **AI Teammates**：20+ 预置职能角色（社媒经理、冷外联、合同审查、竞品监控、简历筛选等），24/7 自主工作
- **共享记忆系统**：纠正一个 Agent，所有 Agent 同步学习；团队级知识积累
- **Slack 集成**：AI 队友以 bot 身份出现在频道，@即可触发
- **自动化**：定时任务、Webhook、GitHub 集成
- **AI 原生文档格式**：支持 Markdown、CSV、HTML 等 AI 友好格式

### 商业模式

采用 **Credits 按用量付费**，拒绝按席位订阅：

| 项目 | 详情 |
|------|------|
| 免费额度 | 注册送 1,000 Credits，无需信用卡 |
| 单价 | $1 = 100 Credits，永不过期 |
| 人数限制 | 无限，只按 AI 工作量收费 |
| 功能分级 | 无，所有功能完全开放 |
| 订阅选项 | Starter $20/月、Standard $200/月（含月度 Credits） |

### 产品亮点

1. **Agent-first 架构** — 整个工作空间围绕 Agent 需求设计（文件系统、记忆、技能、工具链）
2. **共享记忆形成知识复利** — 团队所有 Agent 共享上下文，避免重复教育
3. **Credits 定价降低摩擦** — 全员都能用上 AI，不按席位收费
4. **覆盖 8 大场景**：营销、销售、运营、HR、财务、产品、创作者、Agency

### 风险与不足

1. **极早期**（2026.3 刚发布），产品成熟度待验证
2. **差异化壁垒薄** — Notion AI、飞书 AI 都在跟进 Agent 能力
3. **Credits 消耗不透明** — 用户难预估单次对话成本
4. **Slack 强依赖** — 飞书/钉钉/Teams 用户无法体验核心功能
5. **LLM 成本波动风险** — 按用量收费需自担模型调用成本

### 与我们的关系

| 维度 | Moxt | 我们的方向 |
|------|------|---------|
| 定位 | 通用 AI 工作空间 | 红人协作垂直场景 |
| 目标用户 | 小型创业团队/Solo Creator | 机构、商家、达人三方 |
| AI 角色 | 通用职能 Agent | 红人协作专用（翻译、Brief、排期） |
| 协作模式 | 团队内部协作 | 跨组织双向协作 |
| 行业深度 | 浅（通用平台） | 深（红人营销垂直） |

**启示**：Moxt 验证了"AI-Native Workspace"概念的市场吸引力和 Credits 定价模式的可行性。但它是通用工具，不解决红人协作的具体痛点（跨组织、多语言、Brief 管理、排期协调）。我们可以借鉴其 Agent 架构理念和定价策略，但在垂直场景深度上做差异化。

## 九、差异化方向

核心公式：**Email 级低摩擦 + Grin 级功能 + AI 原生 + 多语言**

- 达人端：Web 应用（邮件邀请 → 点链接 → 立即协作）
- AI 核心场景：
  - 多语言实时翻译与沟通
  - Brief / 脚本自动生成
  - 内容辅助（生图、文案）
  - 沟通摘要与 action items 提取
  - 排期智能建议与冲突检测
- 数据壁垒：合作历史越多，AI 越懂这对关系的偏好
- 跨境优势：中国机构/商家 ↔ 东南亚/欧美达人的协作是核心场景

## 十、方向可行性结论

| 维度 | 判断 | 原因 |
|------|------|------|
| 市场空白 | ✅ 真实存在 | 海外协作层全是 Email 拼凑，无专用工具 |
| 迁移成本 | ✅ 低 | 海外 Email-based 工作流松散，无超级 App 锁定 |
| AI 价值 | ✅ 10x | 多语言跨境场景下翻译+内容生成是刚需 |
| 时机 | ✅ 对 | AI 能力成熟 + TikTok Shop 跨境电商爆发 |
| 团队适配 | ✅ 高 | 自身是机构方，有真实达人和商家资源，懂海外市场 |
| 竞争格局 | ✅ 有空间 | 最近竞品 Grin 无 AI、无多语言、偏贵偏重 |
